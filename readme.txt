# Compile:
$ mkdir build
$ cd build
$ cmake −DCMAKE_BUILD_TYPE=Release ..
$ make -j 4

# Run:
$ ./rt −out <output_path> −scene <scene_number>

Сохраняйте изображения в формате .jpg

При работе была использована информация из открытого источника: https://habr.com/en/post/436790/

Реализовано:
* Базовая часть (+15): 2 примитива (сфера и плоскость)
* Субъективная реалистичность сцены (+2): (мышки)
* Использование поверхностей второго порядка (+4): (эллипсоид)
* Устранение ступенчатости (+1).
* Материалы: зеркало, стекло, просто цвета(черный, желтый, фиолетовый).
